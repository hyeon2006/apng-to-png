# APNG 특정 프레임 추출기

이 스크립트는 지정된 폴더 내의 모든 APNG(또는 PNG) 파일에서 **사용자가 지정한 특정 프레임**을 추출하여, 순차적인 번호(`001.png`, `002.png`...)로 저장합니다.

만약 파일이 움직이지 않는 이미지이거나 지정한 프레임이 존재하지 않을 경우, 자동으로 **첫 번째 프레임**을 추출하여 에러를 방지합니다.

## 📋 주요 기능

* **프레임 선택 가능:** 실행 시 몇 번째 프레임을 추출할지 정수를 입력받습니다. 해당 프레임이 없는 파일은 자동으로 **첫 번째 프레임(Index 0)**을 추출합니다.
* **자동 넘버링:** 파일 이름순으로 정렬 후 `001.png`부터 순서대로 이름을 변경해 저장합니다.
* **자동 폴더 생성:** 출력 폴더가 없으면 자동으로 생성합니다.

## 🛠️ 필수 조건

* Python 3.x
* **Pillow** 라이브러리

```bash
pip install Pillow
```
